Last Fragment est un site web que j'ai entièrement développé en HTML, CSS, JavaScript (avec jQuery) et PHP, conçu comme un outil complet pour gérer mon Jeu de Rôle Papier.

Ce projet a débuté simple, mais il s'est progressivement complexifié. Il est mon outil indispensable pour mes sessions de jeux de rôles papiers

Le site est composé de plusieurs pages, chacune ayant un rôle spécifique :


  • Personnages : Cette section présente la liste de tous les personnages rencontrés par mes joueurs. Les images proviennent de divers artistes en ligne, certaines ont été retouchées, et d'autres sont générées par intelligence artificielle*. En cliquant sur un personnage, des informations détaillées ainsi qu'une description apparaissent. La gestion des personnages est assurée par une base de données, modifiable via un back-office réservé aux administrateurs.
  • Histoire : Ici, quelques points clés du lore sont exposés pour permettre aux nouveaux joueurs de s'immerger dans l'univers avant de rejoindre l'aventure. Certaines pages sont encore verrouillées, car mes joueurs n'ont pas encore exploré toutes les zones disponibles dans ce monde.
  • Outils : Cette section propose deux outils que j'utilise régulièrement. Le premier est un générateur de monstres qui crée des statistiques personnalisées en fonction de mes besoins grâce à des algorithmes aléatoires. Le second est un petit quiz qui révèle le pouvoir magique de naissance que quelqu’un aurait s'il était né dans l'univers de Last Fragment. Je vous invite à faire ce quiz ; aucune donnée n'est stockée, mais connaître votre résultat pourrait susciter des discussions intéressantes !
  • Profil : Accessible uniquement après connexion (un compte invité est disponible), cette page affiche les statistiques du joueur, ses objets, équipements, passifs et compétences. Le joueur peut modifier ses équipements à tout moment.

Enfin, le site cache plusieurs pages secrètes, notamment un back-office permettant de gérer les statistiques, d'ajouter des personnages ou des objets, ainsi qu'une page dédiée à la simulation des combats, qui enregistre également un historique des affrontements lors des sessions de jeu. Il y a aussi des pages d'énigmes que mes joueurs doivent résoudre pour obtenir des informations cruciales pour l'histoire.


Technologies utilisées :

  • HTML
  • CSS
  • JS (+JQuery sur certaines pages)
  • PHP
  • SQL
